The one-month internship at Dyashin Technosoft, Bengaluru provided a valuable and detailed learning experience in the field of full-stack web development using the MERN stack. Through structured training sessions, practical coding exercises, and project-based learning, I gained immense insight into how scalable web applications are architected, developed, and deployed. The internship covered core technologies including HTML, CSS, JavaScript, React.js, Node.js, Express.js, MongoDB, JWT authentication, and file handling with Multer.

The final project, “Internship & Job Portal,” allowed me to apply these theoretical concepts to build a fully functional recruitment platform. This project was designed to bridge the gap between students and recruiters by offering a seamless interface for job posting, searching, and application management. By developing features such as role-based user authentication (Student, Recruiter, Admin), dynamic job feeds, profile management, and resume upload capability, I gained deep practical knowledge of component-based architecture, RESTful API development, and complex database modeling. The integration of a responsive React frontend with a secure Node.js backend demonstrated my ability to deliver a comprehensive full-stack solution.

Overall, the internship significantly strengthened my technical proficiency in the MERN stack, honed my problem-solving skills, and gave me a solid understanding of the software development lifecycle. It has greatly boosted my confidence in building professional, production-ready applications and prepared me for a successful career in software engineering and web development.
